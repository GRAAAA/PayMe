# PayMe

A local-first iOS receipt splitter built with SwiftUI, SwiftData, Vision, and VisionKit.

## Run

Open `PayMe.xcodeproj` in Xcode 16 or newer, select an iPhone simulator or device, choose a development team if running on hardware, and press Run.

Receipt scanning requires a physical iPhone. The app also supports fully manual receipt entry.

## Privacy

OCR runs on-device with Apple Vision. Receipt images are not uploaded or persisted. No API key or backend is required.
